    职位共享是什么？

    • 你可以将职位共享给用人部门的负责人，以及相关的HR，让他们看到你的努力
    • 共享后，该职位收到的简历将自动共享给同事，他们将能够对简历进行评价，甚至参与到该简历的招聘流程
    • 共享后，你的同事可以统一查看意向候选人列表以及面试日程，而不需要你一份一份转发，对方一份一份查看
    • 简招会记录任何对于简历的操作，你们可以任意评价、沟通和协作，这些候选人都是看不到的
    • 这样还可以避免很多事后扯皮有木有~
    • 除此之外，你依旧可以转发单份简历
